# Health Indicator

[![Check Code](https://github.com/Milka79rus/health-indicator/actions/workflows/run.yml/badge.svg)](https://github.com/Milka79rus/health-indicator/actions/workflows/run.yml)

Простая функция для отображения статуса здоровья игрового персонажа в зависимости от количества HP.

## Требования
* 100% покрытие тестами по строкам.
* Интеграция с GitHub Actions.
* Соответствие правилам ESLint.

## Установка и запуск
1. `npm install` — установка зависимостей.
2. `npm test` — запуск тестов и проверка покрытия.
3. `npm run lint` — проверка стиля кода.
